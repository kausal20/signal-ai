These bundled SVG marks are real brand/logo assets, not generated artwork.

- Most brand marks are downloaded from Simple Icons: https://simpleicons.org/
- OpenAI uses the 2025 symbol SVG from Wikimedia Commons: https://commons.wikimedia.org/wiki/File:OpenAI_logo_2025_(symbol).svg
- Runway, Microsoft, and Azure use static SVGs from Lobe Icons: https://github.com/lobehub/lobe-icons

They are rendered in the app through `src/lib/brandLogos.ts`.
